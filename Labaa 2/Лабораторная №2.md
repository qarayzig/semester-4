## Задание

Необходимо было реализовать программу на основе паттерна «Декоратор», которая:

- преобразует данные в формат YAML;
- преобразует данные в формат CSV;

- использует аннотацию типов.

## Реализация интерфейса

В программе был создан базовый интерфейс `Component`, от которого наследуются основной компонент и декораторы.

```python
class Component(ABC):
    """Базовый интерфейс компонента."""

    @abstractmethod
    def operation(self) -> dict[str, Any] | str:
        """Возвращает данные о курсах валют."""
        raise NotImplementedError
```

Также был создан интерфейс `FileSaver`, который описывает метод сохранения данных в файл.

```python
class FileSaver(ABC):
    """Интерфейс для сохранения данных в файл."""

    @abstractmethod
    def save_to_file(self, filename: str) -> None:
        """Сохраняет данные в файл."""
        raise NotImplementedError
```

## Базовый компонент

В качестве базового компонента был реализован класс `CurrencyRatesComponent`.

Он получает данные о курсах валют с помощью API и возвращает их в виде словаря.

```python
class CurrencyRatesComponent(Component):
    """Компонент, получающий курсы валют в формате словаря."""

    API_URL = "https://www.cbr-xml-daily.ru/daily_json.js"

    def operation(self) -> dict[str, Any]:
        """Получает курсы валют через API и возвращает словарь."""
        with urlopen(self.API_URL) as response:
            data = response.read().decode("utf-8")

        return json.loads(data)
```

## Базовый декоратор

Класс `Decorator` реализует общий механизм обертывания компонента.

```python
class Decorator(Component, FileSaver):
    """Базовый класс декоратора."""

    def __init__(self, component: Component) -> None:
        """Инициализирует декоратор компонентом."""
        self._component = component

    @property
    def component(self) -> Component:
        """Возвращает обернутый компонент."""
        return self._component

    def operation(self) -> dict[str, Any] | str:
        """Возвращает результат работы обернутого компонента."""
        return self._component.operation()
```

## JSON-декоратор

Класс `JsonDecorator` преобразует данные в JSON-формат и может сохранять результат в файл `.json`.

```python
class JsonDecorator(Decorator):
    """Декоратор, преобразующий данные в JSON-формат."""

    def operation(self) -> str:
        """Возвращает данные в формате JSON."""
        data = self.component.operation()
        return json.dumps(data, ensure_ascii=False, indent=4)

    def save_to_file(self, filename: str) -> None:
        """Сохраняет данные в JSON-файл."""
        Path(filename).write_text(self.operation(), encoding="utf-8")
```

## YAML-декоратор

Класс `YamlDecorator` преобразует данные в YAML-формат и может сохранять результат в файл `.yaml`.

```python
class YamlDecorator(Decorator):
    """Декоратор, преобразующий данные в YAML-формат."""

    def operation(self) -> str:
        """Возвращает данные в формате YAML."""
        data = self.component.operation()
        return yaml.safe_dump(data, allow_unicode=True, sort_keys=False)

    def save_to_file(self, filename: str) -> None:
        """Сохраняет данные в YAML-файл."""
        Path(filename).write_text(self.operation(), encoding="utf-8")
```

## CSV-декоратор

Класс `CsvDecorator` преобразует данные о валютах в CSV-формат и может сохранять результат в файл `.csv`.

```python
class CsvDecorator(Decorator):
    """Декоратор, преобразующий данные о валютах в CSV-формат."""

    def operation(self) -> str:
        """Возвращает данные о валютах в формате CSV."""
        data = self.component.operation()
        valutes = data.get("Valute", {})

        rows = [["CharCode", "Name", "Nominal", "Value", "Previous"]]

        for valute in valutes.values():
            rows.append(
                [
                    valute.get("CharCode"),
                    valute.get("Name"),
                    valute.get("Nominal"),
                    valute.get("Value"),
                    valute.get("Previous"),
                ]
            )

        return "\n".join(",".join(map(str, row)) for row in rows)
```

## Тестирование

Для проверки работы программы были написаны тесты с использованием библиотеки `unittest`.

Для тестирования был создан тестовый компонент `FakeCurrencyComponent`, который возвращает заранее заданные данные. Это позволяет проверять программу без обращения к интернету.

```python
class FakeCurrencyComponent(Component):
    """Тестовый компонент с заранее заданными курсами валют."""

    def operation(self) -> dict[str, Any]:
        """Возвращает тестовые данные о курсах валют."""
        return {
            "Date": "2026-06-09T11:30:00+03:00",
            "Valute": {
                "USD": {
                    "CharCode": "USD",
                    "Name": "Доллар США",
                    "Nominal": 1,
                    "Value": 78.5,
                    "Previous": 79.1,
                }
            },
        }
```

Для каждого декоратора было написано по два теста:

- проверка результата метода `operation()`;
- проверка сохранения данных в файл.

Пример теста JSON-декоратора:

```python
def test_operation_returns_json(self) -> None:
    """Проверяет, что декоратор возвращает строку JSON."""
    component = JsonDecorator(FakeCurrencyComponent())
    result = component.operation()
    data = json.loads(result)

    self.assertEqual(data["Valute"]["USD"]["CharCode"], "USD")
```

Полный набор тестов находится в файле `test_main.py`.
